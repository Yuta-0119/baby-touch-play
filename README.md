# あかちゃんタッチあそび ✨

生後4ヶ月ごろの赤ちゃん向けの、画面をタッチするとキラキラ光って優しい音が鳴るWebあそびです。

- 画面のどこを触っても反応します（狙う必要なし・複数の指でもOK）
- やさしい鈴の音（ペンタトニック音階）とキラキラのパーティクル演出
- 外部ファイル不要の自己完結HTML（オフラインでも動作）

## 遊ぶ

GitHub Pagesで公開：`play.html`

## 検索エンジンについて

`<meta name="robots" content="noindex">` と `robots.txt` により、検索エンジンにインデックスされない設定にしています。
